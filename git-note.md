## Git Note

### গিটের ভার্শন চেক করতেঃ
```
git --version
```

### গিট কনফিগার করাঃ
```
git config --global user.name "your-name"
git config --global user.email "your-mail-name"
```

### প্রজেক্টে গিট ইনিশিয়ালাইজ(initialized) করাঃ
```
git init
```

### স্ট্যাটাস চেক করাঃ
```
git status
```

### লগ চেক করাঃ
```
git log
```

### ফাইল ট্র্যাক বা এড করাঃ
```
git add filename
```

### সব ফাইল ট্র্যাক বা এড করাঃ
```
git add --all
//or
git add .
```
